# MathEngine
직접 만든 수학 엔진입니다.

# features 
- Constans : 상수 정의

- MathUtils : 자주 사용되는 수학 함수 정의

- Vector2 : 2차원 벡터
- Matrix2x2 : 2x2 행렬

# Dependency
Constans <- MathUtils

MathUtils <- Vector2
MathUtils <- Matrix2x2

Vector <- Matrix2x2
